# PC-VUE-Reception

管理员账号：admin <br>

数据库的配置: 将项目文件内的sql 用navicat 导入即可.<br>
后台运行：进入项目文件夹的serve 依次点击安装.bat，安装nodemon.bat，最后点击运行.bat即可启动后台<br>
项目启动: 进入项目文件夹的cilent_home 依次点击安装.bat，安装sass.bat，最后点击运行.bat即可启动项目文件<br>
